[Deprecated] 关于 | About

jsMind 是一个显示/编辑思维导图的纯 javascript 类库，其基于 html5 的 canvas 或 svg 实现。jsMind 以 BSD 协议开源，在此基础上你可以在你的项目上任意使用。你可以在此浏览适用于 jsMind 的 BSD 许可协议(中英文版本)。

jsMind is a pure javascript library for mindmap, it base on html5 canvas or svg. jsMind was released under BSD license, you can embed it in any project, if only you observe the license. You can read the BSD license agreement for jsMind in English and Chinese version here.

[Deprecated] 用法 | Simple Usage

This is just a simple usage demo below:

<html>
    <head>
        <link type="text/css" rel="stylesheet" href="style/jsmind.css" />
        <script type="text/javascript" src="js/jsmind.js"></script>
        <!--
        <script type="text/javascript" src="js/jsmind.draggable.js"></script>
        -->
    </head>
    <body>
        <div id="jsmind_container"></div>

        <script type="text/javascript">
            var mind = {
                // 3 data format supported ...
                // see Documents for more information
            };
            var options = {
                container:'jsmind_container',
                editable:true,
                theme:'orange'
            };
            var jm = new jsMind(options);
            jm.show(mind);
        </script>
    </body>
</html>
